## Yapılacaklar
- repo nasıl çalışıyor anla(sor gemini'a anlatsın).
- db üzerinde bir table oluştur adı auth olack şekilde. 
- table'ı datalayer kısmında orm yapısını kurarak register et.
- repository methodları kullanarak service klasörü içerinde auth_service adlı bir python dosyası oluştur ve içersine post endpointleirnin çalışması için service yapısını kur.
- config dosyasını api'nin ayarlar kısmı olarak düşün ve ayarlarla ilgili kısımları orda hallet
- .env diye bir dosyayı ana dizinde oluşturup database credentialları orda tanımla şunun gibi:

DATABASE_HOST = 
DATABASE_NAME = 
DATABASE_USERNAME =
DATABASE_PASSWORD = 

- database bağlantısının yapıldığından emin ol.
